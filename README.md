# TPI-programacion-
# TPI - Gestión de Países en Python

Trabajo Práctico Integrador de Programación 1 - UTN Tecnicatura Universitaria en Programación a Distancia.

## Descripción

Aplicación en Python que permite gestionar información sobre países desde un archivo CSV. Ofrece un menú en consola con opciones para agregar, actualizar, buscar, filtrar, ordenar países y ver estadísticas.

## Estructura del proyecto

```
TPI-programacion/
│
├── paises.csv          → Dataset base de países
├── main.py             → Menú principal
├── archivos.py         → Lectura y escritura del CSV
├── gestor.py           → Agregar y actualizar países
├── busqueda.py         → Buscar y filtrar países
├── ordenamiento.py     → Ordenar países
├── estadisticas.py     → Estadísticas generales
└── README.md           → Este archivo
```

## Instrucciones de uso

1. Cloná o descargá el repositorio
2. Asegurate de tener Python 3.x instalado
3. Ejecutá el programa con:

```
python main.py
```

4. Seguí las opciones del menú en consola
